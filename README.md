# バイオインテグレーション学会 WEBサイト

静的サイトです。`index.html` を入口として、`members.html` と `messages.html` に遷移します。

## 公開方法

このリポジトリは GitHub Pages で公開できます。

1. GitHub で新しいリポジトリを作成
2. このフォルダをそのリポジトリへ push
3. GitHub の `Settings` -> `Pages` を開く
4. `Source` で `GitHub Actions` を選択
5. `main` ブランチへ push すると自動で公開

公開URLは通常、以下の形式です。

`https://<GitHubユーザー名>.github.io/<リポジトリ名>/`

## ファイル構成

- `index.html`: トップページ
- `members.html`: メンバー紹介
- `messages.html`: ごあいさつ
- `PIC/`: 画像素材
